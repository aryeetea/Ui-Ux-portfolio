# UI/UX Portfolio

Portfolio site for Aileen Naa Ayele Aryeetey, a UI/UX designer and product builder.

## Overview

This is a static portfolio website built with plain HTML, CSS, and JavaScript. It highlights featured work, case studies, an about section, and contact information.

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` - page structure and content
- `style.css` - visual styling and layout
- `script.js` - interactive behavior

## Local Preview

Open `index.html` in a browser, or serve the folder with any simple static server.

## Deployment

This repository includes a GitHub Actions workflow that deploys the site to GitHub Pages.

After GitHub Pages is enabled for this repository, pushes to `main` will publish the site automatically.

## Notes

- The page references `photo.jpg` for the hero and about images. Add that file at the repo root if you want the live site to show your photo instead of the fallback initial.
